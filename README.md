# プロジェクト管理リスト（SharePointカスタムビュー）

> SharePointリストをカスタムJSONフォーマットでギャラリービューに整形したテンプレート
> プロジェクトをカード形式で表示し、進捗をビジュアルで確認できる

---

## 解決した課題

- 標準のSharePointリストではプロジェクトの進捗状況が一目でわからなかった
- カスタムJSONフォーマットを適用し、**カード形式＋プログレスバー**で視認性を改善

---

## カードの表示内容

| 表示項目 | フィールド | 内容 |
|---|---|---|
| アイコン | Icon | プロジェクトを象徴する絵文字など |
| ステータス | Status | 未着手 / 進行中 / 完了 |
| タイトル | Title | プロジェクト名（2行まで表示） |
| 進捗バー | Progress | 数値（%）でバーの長さが変わる |
| リンク | Link | クリックで関連ページへ遷移 |

---

## ステータスの色分け

| ステータス | 色 |
|---|---|
| 未着手 | 🔴 赤（#D13438） |

## 使い方

### 1. SharePointリストのフィールドを準備する

| フィールド名 | 内部名 | 種類 |
|---|---|---|
| タイトル | Title | テキスト |
| アイコン | Icon | テキスト（絵文字を入力） |
| ステータス | Status | 選択肢（未着手／進行中／完了） |
| 進捗 | Progress | 数値（0〜100） |
| リンク | Link | ハイパーリンク |

### 2. ギャラリービューに適用する

1. SharePointリストを開く
2. 右上の **「ビューの切り替え」→「ギャラリー」** を選択
3. 右上の ⚙ **「現在のビューの書式設定」** をクリック
4. **「詳細モード」** を開く
5. `gallery-view.json` の内容を貼り付けて **「保存」**

---

## 工夫したポイント

- ステータスに応じて**バッジの色を自動変更**（if式で条件分岐）
- 進捗数値に応じて**プログレスバーの色と長さを自動変更**
- カードをクリックすると**関連ページへ直接遷移**できるリンク設計
- カード幅210px・高さ150pxのコンパクトサイズで一覧性を確保
